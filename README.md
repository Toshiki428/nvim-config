# Neovim Configuration

## 概要

このリポジトリは、個人的なNeovim設定を管理しています。

日々の開発作業をより効率的かつ快適に行うためにカスタマイズを続けます。

## 特徴

- Lazy.nvim を使用したプラグイン管理
- LSP (Language Server Protocol) によるコード補完と診断
- ファイルツリーを使用したナビゲーション機能
- テーマによる視覚的なカスタマイズ

## 導入方法

1. このリポジトリをクローンします。
   ```bash
   git clone https://github.com/Toshiki428/nvim-config.git ~/.config/nvim
   ```
2. Neovimを起動すると、Lazy.nvimが自動的にプラグインをインストールします。

## プラグイン

主要なプラグインは `lua/plugins/` ディレクトリに定義しています。
